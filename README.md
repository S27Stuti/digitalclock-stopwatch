# ⏰ Digital Clock & Stopwatch App

A modern and responsive ReactJS application that combines a **Digital Clock** and **Stopwatch** in a single interface. The application displays the current time in real-time and provides stopwatch functionality with Start, Pause, and Reset controls.

## 🚀 Live Demo

https://brilliant-croissant-5fbf60.netlify.app/

## ✨ Features

* Real-time Digital Clock
* Live time updates every second
* Stopwatch functionality
* Start, Pause, and Reset controls
* Responsive design for Desktop, Tablet, and Mobile
* Modern and clean UI
* Built using React Hooks

## 🛠️ Technologies Used

* ReactJS
* JavaScript (ES6+)
* HTML5
* CSS3
* React Hooks (`useState`, `useEffect`, `useRef`)

## ⚙️ Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project folder

```bash
cd digital-clock-stopwatch
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm start
```

5. Open your browser

```bash
http://localhost:3000
```

## 🎯 Functionality

### Digital Clock

* Displays the current system time.
* Updates automatically every second.

### Stopwatch

* Start timer
* Pause timer
* Reset timer
* Displays time in HH:MM:SS format
